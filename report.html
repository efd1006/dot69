dot69 Programming Language
Features
  • Not Case sensitive
  • You don’t need to worry about semicolon
  • Similar to BASIC and PHP
  • Syntax is based on tagalog words
Syntax
To Compile: Go to the folder where compiler.py is then type:
py compiler.py filename
To print an output:
To print an output: <ilabas> <"String"> Example: ilabas "string" ILABAS "string"
To display arithmetic operations: <ilabas> term + term Example ilabas 4+5 ILABAS 4+5
To store a variable: $<varname> = <value> Example: $var = "a" $var = 1 $var = 1+2
To get the input: <ipasok> "string" $<variablename> Example: ipasok "What is your name: " $name
If statement: pag <condition> ituloy // code punasan Example: pag 1 == 1 ituloy ilabas "true" punasan
